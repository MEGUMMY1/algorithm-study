# 문제-1085 직사각형에서 탈출

- 이 문제는 단순 수학 문제이다. 그림으로 상상해보면 매우 쉬워진다.
- 경계선으로 가는 최단 거리는 전체 길이에서 현재 길이를 뺐을 때, 더 작은 것이다.
- 전체 가로 길이인 w에서 x를 뺴준 후, x와 비교.
- 전체 세로 길이인 h에서 y를 뺴준 후, y와 비교. 

다음 규칙을 구현해준다.

```C
int x, y, w, h, m_x, m_y;
cin >> x >> y >> w >> h;
m_x = x >= w - x ? w - x : x; //삼항 연산자 이용
m_y = y >= h - y ? h - y : y;
cout << (m_x > m_y ? m_y : m_x);
```
